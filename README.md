# 🔗 URL Shortener

A simple Python application that shortens long URLs using the TinyURL API.

## ✨ Features

- Shorten any valid URL
- Fast and lightweight
- Uses the TinyURL API
- Beginner-friendly Python project

## 📦 Installation

```bash
git clone https://github.com/your-username/url-shortener.git
cd url-shortener
pip install -r requirements.txt
```

## ▶️ Run

```bash
python main.py
```

## 💻 Example

Input:

```
https://www.google.com
```

Output:

```
https://tinyurl.com/xxxxx
```

## 🛠 Tech Stack

- Python
- Requests
- TinyURL API
